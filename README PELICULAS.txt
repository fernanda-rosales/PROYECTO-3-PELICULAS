Para correr el programa TiendaPeliculas se debe de:

instalar el IDE netbeans 8.2 , pero antes debe tener instalado en el 
sistema la actualización 19 de Java SE Development Kit (JDK) 

Si no tiene una instalación de JDK, no puede continuar con la instalación del IDE,
posteriormente ejecutar el archivo exe y tendra intalado el id donde fue creada
nuestra aplicacion y donde podra ser ejecutada una ves descompromiendo la carpeta
de TiendaPeliculas.zip que se anexa.

abrir el netbeans y dar click en file, despues open project y buscar la carpeta donde
se encuentra el archivo TiendaPeliculas una vez abierto dar click en run project y se
compilara el proyecto teniendo su ejecucion, el programa maneja base de datos.


Por lo tanto se tendra que instalar el manejador de base de datos oracle express edition ya
que se creo una mini base de datosa para el desarrollo del programa y adjuntamos
el scrip bdatos para poder abrir la base, inluyendo el link de descarga del programa
que fue cargado a nuestros repositorios de github
